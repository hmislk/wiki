Here, all the available membership schemes will be listed. To add a new membership scheme, select "Add" and then fill in the details, such as:
* Name
* Code
* Points for 1000
* Discount Scheme

Then select "save" to add a new membership scheme. To remove a membership scheme, select it and then select "Delete."
